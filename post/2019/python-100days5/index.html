水仙花數(Narcissistic number)

用來描述一個 N 位非負數, 其中每個數字的 N 次方和等於該數字本身 代碼:

    for num in range(1000):
        squares = len(str(num))
        count = 0
        count = sum([int(x)**squares for x in str(num)])
        if num == count:
            print(f"The Narcissistic number is : {count}" )

完美數(Perfect number)

用來描述一個數字, 除了自己以外的所有真因子的總和, 等於它自己, 就是完美數 代碼:

    for num in range(100):
        list = [x for x in range(1, num) if num%x==0]
        if sum != [] and sum(list) == num:
            print(num)
